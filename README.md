# idr-frontend-project
# Institute of Digital Risk (IDR) Website

This project is a simple responsive website created for the **Institute of Digital Risk (IDR)**.
It is a single-page website that explains the mission of the institute and the services it provides.

The website was built using **HTML, CSS and JavaScript** without any CSS frameworks.

---

## About the Project

The **Institute of Digital Risk (IDR)** is an industry-led institute focused on training professionals in digital risk, cyber security and technology risk.

This website includes information about:

* The mission of IDR
* The services and programs offered
* The community it supports
* A contact form for users to register interest

---

## Features

* Responsive design (works on desktop and mobile)
* Sticky navigation bar
* Smooth scrolling between sections
* Hero section with call-to-action buttons
* Service cards explaining the IDR model
* Contact form
* Clean and simple user interface

---

## Sections in the Website

1. **Hero Section**
   Introduction to the Institute of Digital Risk.

2. **About Section**
   Information about the institute and its purpose.

3. **Services Section**
   Three main pillars of IDR:

   * Academy
   * Innovation & Incubation
   * Advisory Services

4. **Community Section**
   Describes who the institute serves.

5. **Contact Section**
   Simple form to register interest.
